# package grafonnet

Jsonnet library for rendering Grafana resources

## Install

```
jb install github.com/grafana/grafonnet/gen/grafonnet-v9.4.0@main
```

## Usage

```jsonnet
local grafonnet = import "github.com/grafana/grafonnet/gen/grafonnet-v9.4.0/main.libsonnet"
```
